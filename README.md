# Registro de Pensamentos — TCC

App PWA para exercício de Registro de Pensamentos da Terapia Cognitivo-Comportamental.

## Como publicar no GitHub Pages

```bash
# 1. Clone o repositório que você criou
git clone git@github.com:vinnitog/RDP.git
cd RDP

# 2. Copie todos estes arquivos para dentro da pasta:
#    index.html, manifest.json, sw.js, icon-192.png, icon-512.png

# 3. Faça o commit e push
git add .
git commit -m "primeiro deploy"
git push origin main

# 4. Ative o GitHub Pages:
#    → Repositório no GitHub > Settings > Pages
#    → Source: "Deploy from a branch"
#    → Branch: main / (root)
#    → Save

# 5. Aguarde ~1 minuto. Seu app estará em:
#    https://vinnitog.github.io/RDP/
```

## Como instalar como app no Android

1. Abra `https://vinnitog.github.io/RDP/` no **Chrome**
2. Toque no menu ⋮ (três pontos, canto superior direito)
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — o ícone aparece na tela inicial como um app nativo
